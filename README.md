# Linarya RP — Site vitrine

Mini landing page légère et responsive pour le serveur FiveM **Linarya**. Le site met en avant l’accueil, le règlement (placeholder), un lexique RP et une section contact, le tout avec un mode sombre/claire basculable.

## Structure

- `index.html` — page unique avec navigation ancrée.
- `styles.css` — thème flat design, responsive et toggle clair/sombre.
- `script.js` — gestion du thème, menu mobile et horodatage du footer.
- `lexique.txt` — contenu source pour la section Lexique.

## Prérequis & lancement

Aucun build requis : ouvrez simplement `index.html` dans votre navigateur.  
Pour éditer les styles ou le script, modifiez directement `styles.css` et `script.js`.

## Personnalisation

- Remplacez le lien Discord (section Contact) dès qu’il est disponible.
- Mettez à jour la section Règlement lorsque le contenu sera prêt.
- Adaptez facilement couleurs et espacements via les variables CSS en tête de `styles.css`.
